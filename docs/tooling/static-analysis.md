# Static Analysis

Static analysis tools for Windows kernel driver vulnerability research.

## Binary Diffing

- **BinDiff** — Google's binary diffing tool for matched function comparison
- **Diaphora** — Open-source binary diffing plugin for IDA Pro
- **Ghidra Version Tracking** — Built-in Ghidra function matching

## Decompilation

- **Ghidra** — NSA's open-source decompiler with excellent Windows driver support
- **IDA Pro / Hex-Rays** — Industry standard decompiler
- **Binary Ninja** — Modern decompiler with good API

## Pattern Matching

- **Semgrep** — Pattern matching on decompiled C output
- **YARA** — Pattern matching on binary code
- **AutoPiff** — Automated patch diffing with semantic rule matching
