# Fuzzing

Kernel driver fuzzing frameworks and techniques.

## IOCTL Fuzzing

- **kAFL** — Intel PT-based kernel fuzzer
- **IOCTLpus** — Windows IOCTL fuzzer
- **DIFUZE** — Android/Linux driver interface fuzzer (concepts applicable)

## Coverage-Guided

- **WTF (What The Fuzz)** — Snapshot-based Windows kernel fuzzing
- **Hyntrospect** — Hyper-V and kernel component fuzzer

## Harness-Based

- **LibFuzzer** — Compile driver code as user-mode harness
- **Custom harnesses** — Extract driver logic for targeted fuzzing
